새 글 또는 새 글 초안 생성

```shell
hexo new [post | draft] <title>
```

draft 초안 발행

```
hexo publish <title>
```

서버 실행

```shell
hexo server
```

| 옵션  | 설명              |
| --------- | ----------------- |
| `--draft` | draft 포스트 표시 |

정적 파일 생성

```
hexo generate
```

배포

```shell
hexo deploy
```

| 옵션              | 설명                  |
| ----------------- | --------------------- |
| `-g`, `-generate` | 배포 전 generate 수행 |
